# The Vagabond — Narrative Sticker Catalogue

These are companion stickers, not enlarged UI emoji. Each must use a 512px
transparent composition, one recognisable scene, bold native-vector planes,
and a readable hold after its action.

## Visual rule

Recurring commanders use graphite and cobalt expedition armour, an ivory
faceplate, cyan signal light, violet crystal energy, gold only for achievement,
and danger red only for threats. Never substitute a generic UI symbol for the
scene described above.
