# The Vagabond — Character & Reaction Stickers, V2

Six native-transparent Telegram TGS stickers. Each scene is a real character
reaction for chat; none is a UI icon enlarged into a sticker.

| Asset | Chat meaning | Subject and motion |
|---|---|---|
| `commander_acknowledges` | understood / acknowledged | Armoured field commander salutes; two gold signal glints answer from the visor. |
| `crystal_joy` | rare find / celebration | Commander cradles a violet crystal as refractive stars pulse around its facets. |
| `raid_ready` | let’s raid | Commander points forward along a gold route; the red target acquires the destination. |
| `field_medic_recovery` | heal up / recovering | Field medic patch pulses with a precise white-red medical cross and cyan repair glint. |
| `defiant_survivor` | we stand / rebellion | Hooded rebel raises a clenched fist beside a moving crimson faction banner. |
| `survivor_bond` | welcome / together | Two differently marked commanders meet at a gold forearm clasp. |

## Delivery guardrails

- 512 × 512, 60 FPS, 2.0-second transparent loop.
- Native Lottie shapes only; no raster art, masks, effects, or expressions.
- Generated from `assets/visual-system/pipeline/build_vagabond_stickers_v2_characters.py`.
- Structural TGS validation has passed for every file. These are production
  assets; Telegram review/upload is a batch hand-off step, not an iteration
  gate for each individual sticker.
