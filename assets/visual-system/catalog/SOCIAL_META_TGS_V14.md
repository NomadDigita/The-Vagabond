# Vagabond Social & Meta TGS V14

This family extends the approved crystal V14 native-vector visual standard. Each item is a 512px, 60fps, two-second, transparent TGS composition with only native Lottie shapes.

| Key | Real game meaning | Subject-led silhouette | Motion |
|---|---|---|---|
| `federation` | Federation/alliance system | Two interlocking defensive shields and alliance link | Shields answer each other; link energizes |
| `missions` | Mission contracts/objectives | Field contract board and completion mark | Task lines energize, then the completion check draws |
| `destinations` | Destination selection/navigation | Orbital destination globe and waypoint pin | Orbit sweeps; pin pulses |
| `settings` | Settings and calibration | Three physical control rails and sliders | Each slider calibrates independently |
| `feedback` | Player feedback uplink | Transmission bubble with three indicator lights | Lights relay left-to-right and uplink flashes |
| `profile_stats` | Commander profile and stats | Rank badge, commander crest, stat bars | Bars rise and fall as a stats readout |

Outputs are located under `assets/visual-system/animated/<key>_tgs_v14/`. Build them with:

```powershell
python assets/visual-system/pipeline/build_vagabond_social_tgs_v14.py
```
